# 이박사 향기품 만세력 v3

한국 역사적 표준시 + 써머타임 + VSOP87 정밀 절기 계산 만세력 앱

## v3 새 기능
1. **보정값 상세 표시** — 법정시/써머타임/경도보정/균시차 자동 표기
2. **경계 경고** — 절입 ±2시간, 시주 ±30분 경계 자동 감지
3. **정밀검증 모드** — 벽시계 vs 진태양시 비교 (체크박스 토글)
4. **VSOP87 절기** — ephem 라이브러리로 KASI ±1분 정밀도
5. **모바일 최적화** — 글씨 크기 확대, 핀치줌 허용

## 배포
```
streamlit run app.py
```

## 파일 구조
- `app.py` — 메인 앱
- `korea_tz_history.py` — 역사적 표준시/DST 모듈
- `requirements.txt` — 의존성
